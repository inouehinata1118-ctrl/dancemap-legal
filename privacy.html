プライバシーポリシー

DanceMap

制定日：2026年6月11日／最終改定：2026年6月28日

DanceMap運営事務局（以下「当社」といいます）は、当社が提供するダンスイベント情報アプリ「DanceMap」および関連サービス（以下「本サービス」といいます）における、ユーザーの個人情報・利用者情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

第1条（取得する情報）

当社は、本サービスの提供にあたり、以下の情報を取得します。

  • アカウント情報：Sign in with Apple による認証時に取得するメールアドレスおよび Apple が発行する識別子
  • プロフィール情報：ニックネーム、自己紹介、ダンスジャンル、経歴、プロフィール画像、活動エリア、Instagram・YouTube 等のリンク、プロフィールの公開／非公開設定
  • 投稿コンテンツ：ユーザーが登録するイベント情報・フライヤー画像・イベントシリーズ情報、いいね、保存（ブックマーク）、参加記録、フレンド関係、通報・ブロックの情報
  • 位置情報：周辺のイベントを地図上に表示するための端末の位置情報（許可した場合のみ取得し、許可しなくても本サービスの閲覧は可能です）
  • プッシュ通知トークン：保存したイベントの中止通知など、通知の送信に用いる端末識別子（Apple Push Notification service / Firebase Cloud Messaging）
  • 利用状況・端末情報：アプリの利用ログ、アクセス状況、端末の種類、OSバージョン等
  • 購入情報：任意の「チップ（運営への応援）」等のアプリ内課金に関する取引記録

アプリ内課金の決済は Apple（App Store）が処理し、当社がクレジットカード番号その他の決済情報を取得・保管することはありません。当社が Apple から受け取る情報にカード情報は含まれません。

第2条（利用目的）

  • 本サービスの提供・運営・ユーザー認証のため
  • イベントの検索・表示、地図表示、プロフィール表示、フレンド機能、いいね・保存等の各機能を提供するため
  • 保存したイベントの中止のお知らせ等、プッシュ通知を送信するため
  • お問い合わせ対応、通報・ブロック等の不適切な利用への対応のため
  • 本サービスの改善、新機能の開発、不具合の調査・対応のため（アクセス分析を含む）
  • 利用規約に違反する行為への対応のため

第3条（第三者提供・公開される情報）

当社は、法令に基づく場合または人の生命・身体・財産の保護のために必要な場合を除き、あらかじめユーザーの同意を得ることなく、個人情報を第三者に提供しません。

ただし、ユーザーが本サービス上に登録・公開した情報（イベント情報、プロフィール、いいね等）は、本サービスの性質上、他のユーザーが閲覧できる状態となります。プロフィールを「非公開」に設定した場合、フレンド以外のユーザーには、ニックネーム・アイコン・活動エリア・ジャンル・自己紹介などの基本情報のみが表示され、経歴等の詳細はフレンドに対してのみ公開されます。

第4条（外部サービスの利用）

当社は、本サービスの運営にあたり以下の外部サービスを利用しており、これらにデータの保管・処理を委託しています。

  • Google Firebase（Authentication / Cloud Firestore / Cloud Storage / Cloud Functions / Cloud Messaging / Analytics）：認証、データ保管、通知配信、利用状況の分析に利用
  • Apple（Sign in with Apple、Apple Push Notification service、App Store のアプリ内課金）：認証、通知配信、およびアプリ内課金の決済処理に利用

各サービスにおける情報の取り扱いは、各社のプライバシーポリシーに従います。なお、本サービスは広告配信やアプリ横断的なトラッキングを目的とした情報の利用は行いません。

第5条（情報の保管期間・削除）

当社は、利用目的の達成に必要な期間、情報を保管します。ユーザーは、アプリ内の「設定 > アカウントを削除」からいつでもアカウントおよび個人データの削除を行うことができます。削除を行った場合、プロフィール・いいね・保存・参加記録・フレンド関係・通知トークン等の個人データは削除されます。ただし、他のユーザーの閲覧に供されている投稿コンテンツ（イベント情報等）は、サービス運営上、削除されずに残る場合があります。

第6条（位置情報の取り扱い）

本サービスは、周辺のダンスイベントを地図上に表示する目的で、ユーザーが許可した場合に限り端末の位置情報を利用します。位置情報の利用許可は、ユーザーが端末の設定からいつでも変更・取り消すことができます。

第7条（プッシュ通知）

本サービスは、保存したイベントの中止のお知らせ等を目的として、ユーザーが許可した場合に限りプッシュ通知を送信します。通知の許可は、ユーザーが端末の設定からいつでも変更・取り消すことができます。

第8条（安全管理措置）

当社は、個人情報の漏えい、滅失またはき損の防止その他の安全管理のために、必要かつ適切な措置を講じます。委託先に対しても、必要かつ適切な監督を行います。

第9条（お子様の利用）

未成年者が本サービスを利用する場合は、法定代理人（親権者）の同意を得たうえでご利用ください。

第10条（開示・訂正・削除等の請求）

ユーザーは、当社が保有する自己の個人情報について、開示・訂正・利用停止・削除等を請求することができます。請求を希望される場合は、第12条のお問い合わせ先までご連絡ください。本人確認のうえ、法令に従い対応します。

第11条（本ポリシーの変更）

当社は、必要に応じて本ポリシーを変更することがあります。重要な変更を行う場合は、本サービス上での掲示またはアプリ内通知により周知します。変更後に本サービスを継続して利用した場合、変更後の本ポリシーに同意したものとみなします。
